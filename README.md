# SmartLearn - Plataforma de Micro-Learning

Una plataforma web diseñada para optimizar el tiempo de estudio de estudiantes universitarios mediante el uso de Flashcards y el algoritmo de repetición espaciada (SM-2).

## Tecnologías Utilizadas
Este proyecto utiliza una arquitectura basada en un Monorepo (Frontend y Backend separados).

*Frontend:*
* React.js (Vite) - El compilador de react no esta habilitado en esta plantila debido a su impacto en el rendimiento de desarrollo y compilación. Para agregarlo, consulte esta documentación. https://react.dev/learn/react-compiler/installation
* Tailwind CSS / Bootstrap
* HTML5 & CSS3

*Backend:*
* Node.js
* Express.js
* Base de Datos: PostgreQL
